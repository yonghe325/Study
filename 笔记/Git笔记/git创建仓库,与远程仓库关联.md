# diff 查看差异
## 默认工作,暂存区
## 生成一个哈希值
## HEAD:比较工作区,版本库之间的差异
## (/) ;比较工作区,暂存区
## --cached :比较暂存区,版本库
## (版本号) (版本号):两个版本间的差异
## HEAD~ HEAD:当前与上一个版本间的差异
## HEAD~(x) HEAD:当前与提交之前的第 x 个版本间的差异
## (文件名) : 查看这个文件的差异
## (分支号) (分支号):查看两个分支的差异
